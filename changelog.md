# Changelog

All notable changes to Crochet Workspace will be documented here.

## [Unreleased]

## 2026-06-12

- Added project-specific workspace loading so switching projects restores the project’s row counter, timer, notes, pattern sources, and pattern tracker.
- Added hover and pressed-state feedback across page buttons for a more responsive app feel.
- Added Project Timer v1 with quick/project timer modes, start/pause, end session, reset session, clear all time, total time, and recent sessions.
- Added a Current Project tooltip explaining Row Counter vs Pattern Tracker use, and fixed tooltip positioning.
- Disconnected Pattern Tracker row highlighting from the Row Counter so each tool can track independently.
- Added strikethrough styling for completed Pattern Tracker rows.
- Improved Pattern Tracker with next-row highlighting, Uncheck Rows, Delete Pattern, and completion feedback.
- Added Pattern Tracker v1 with pasted row import, checkboxes, progress tracking, and saved tracker state.

## 2026-06-11

- Added Quick Counter and Project Counter modes so the row counter can work standalone or with an active project.
- Refined the Current Project card layout, backup controls, and project actions.
- Added hook-size selector with custom hook option, project status selector, and Materials field.
- Added all-projects backup export alongside current-project backup.
- Added project snapshots and project backup/export support.
- Updated app icons and labels for a friendlier crochet-focused interface.
- Added Clear buttons for Video, Web Pattern, and PDF pattern sources.
- Added PDF Pattern support with PDF URL loading and local PDF upload.
- Added Web Pattern tab support for loading pattern webpages with an open-in-new-tab fallback.
- Added Video Tutorial tab and main dashboard layout.
- Added working voice counter controls for row counting.

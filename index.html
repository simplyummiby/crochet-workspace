🧶 How to Use Crochet Workspace

Three simple steps: start a project, keep your pattern nearby, then track rows and time.

1
Create or open a project

Use Current Project to save the project name, hook, materials, notes, row count, timer history, and pattern links.

2
Add the pattern you are following

Use Pattern Source for a video, webpage, or PDF. If you have written rows, paste them into Pattern Tracker.

3
Track progress as you crochet

Use Quick modes for one-off counting or timing. Switch to Project modes when you want rows and time saved with the active project.

💾 Storage Notice Crochet Workspace currently saves projects, notes, row counts, timers, tutorial notes, and pattern data in this browser. If you clear browser data, switch browsers, or use a different device, your saved workspace may not be there. Use Advanced: Backup & Restore regularly for important projects.

Tip: Row Counter and Pattern Tracker are independent. Use Row Counter for a simple number, Pattern Tracker for row-by-row instructions, or both.

🧶 How to Use Crochet Workspace

A quick guide to projects, patterns, rows, time, notes, and backups.

1. Create a project

Save the project name, hook size, materials, status, notes, rows, timer data, and pattern sources.

2. Add a pattern source

Use 🎥 Video Tutorial, 🌐 Web Pattern, or 📄 PDF Pattern to keep your instructions nearby.

3. Track your rows

Use the Row Counter for manual tracking, or paste rows into the Pattern Tracker and check them off.

4. Track your time

Start a Project Timer session when you crochet. End the session to save it to the project history.

5. Keep notes

Use Project Notes for yarn changes, modifications, reminders, sizing notes, or anything you want to remember.

6. Back up your work

Use Backup & Restore to download the current project or all projects as a JSON backup file.

Tip: Row Counter and Pattern Tracker are independent tools. Use either one—or both—depending on how you like to follow a pattern.
